# Resume

Resume built with LaTeX [(tectonic)](https://tectonic-typesetting.github.io/).

[Resume PDF](https://mrzzy.github.io/resume/assets/pdfs/main-1.pdf)

## Setup

Install [tectonic](https://tectonic-typesetting.github.io/) to typeset the resume.

## Usage

1. Update `src/content.json`
2. `make build` to build the resume content.

> Steps are automated by the Github Actions CI, which attaches the built PDF as build artifact.
